# Home Steward Agent — 推荐模型配置
#
# 用法: ollama pull <model-name>
# 系统首次启动时会根据你的硬件自动探测并推荐模型。
# 你也可以手动选择以下模型之一:
#
# ┌──────────────┬──────────┬────────────────┬────────────────────┐
# │ 模型          │ 参数量   │ 最低配置       │ 推荐场景            │
# ├──────────────┼──────────┼────────────────┼────────────────────┤
# │ qwen3:3b     │ 3B       │ 4GB RAM        │ 树莓派 5 / 低配设备 │
# │ qwen3:7b     │ 7B       │ 6GB RAM        │ 平衡速度与智能       │
# │ qwen3:14b    │ 14B      │ 12GB RAM / 8GB │ 有 GPU 的用户        │
# │              │          │ VRAM           │                     │
# │ qwen3.5:9b   │ 9B       │ 8GB RAM        │ 推理模型(带思考)     │
# │ qwen3:35b    │ 35B      │ 24GB VRAM      │ RTX 3090/4090       │
# └──────────────┴──────────┴────────────────┴────────────────────┘
#
# 推荐:
#   第一次使用 → qwen3:7b
#   有 NVIDIA 显卡 → qwen3:14b 或 qwen3.5:9b
#   只想最快体验 → qwen3:3b
